# VoxMed

VoxMed is an AI-powered web application that enables continuous, non-invasive health monitoring through daily voice recordings. 
Users record a 15-second voice sample each day, which the system analyses in real time to extract six key biomarkers: tremor score, breathlessness score, pitch mean, pitch variation, speech rate, and pause-pattern score. 
These biomarkers are aggregated into an overall health score and stored securely in a cloud database (Supabase).
Anomaly detection logic flags unusual patterns and generates smart health alerts, while interactive dashboards visualize longitudinal trends.
 The system targets chronic-disease management for conditions such as Parkinson's, asthma, depression, post-stroke recovery, and cardiovascular disease — providing an early-warning layer that costs nothing beyond an internet connection.
